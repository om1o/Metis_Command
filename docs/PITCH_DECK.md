# Metis — 10-Slide VC Pitch Deck

A ready-to-build-in-Figma outline. Each slide includes the single sentence that must stick and the talking points underneath.

---

## 1. Title
**Metis — the first local-first autonomous AI operating system.**
- Tagline: "Your second brain. Offline. Forever yours."
- Founder name, Miami, 2026.

## 2. The Problem
**Every major AI runs in someone else's cloud.**
- Enterprise data keeps leaking into OpenAI, Anthropic, and Google training sets.
- Every API call is a tax. Power users hit rate limits daily.
- Cloud AI forgets who you are the second you close the tab.

## 3. The Metis Solution
**A desktop-native, multi-agent AI that runs 100% on your hardware.**
- Tri-core local brain — Qwen 3.6, GLM 5.1, DeepSeek V2 — via Ollama.
- Cognitive memory loop — semantic recall, nightly synthesis, per-day identity updates.
- Docker-sandboxed skill forge — writes and validates its own tools.

## 4. Architecture (diagram)
- Claude.ai-style 3-column UI · Codex agent panel · Artifacts pane.
- Manager → Coder / Thinker / Scholar / Researcher via CrewAI hierarchical.
- Computer Use tools (mss, pyautogui) let Metis see and act on the desktop.

## 5. Tri-Core Engine
**We don't bet on one model. We route.**
- Coder: Qwen 3.6 — best local coding model on edge hardware.
- Scholar: GLM 5.1 — dominant long-context document reader.
- Thinker: DeepSeek V2 — raw reasoning and architecture decisions.

## 6. Market
**$250B+ AI productivity market — almost none of it offline.**
- Wall Street + Big Law + Defense: cannot legally use cloud AI with sensitive data.
- Power developers: already burning $100–500/month on ChatGPT/Claude.
- Privacy-aware consumers: growing fast post-2025 policy shifts.

## 7. Competitive Moat
**Four kill-shots vs. Claude 4.7 / GPT-5 / Gemini:**
1. **Zero-latency desktop** — native hotkey, computer use, screen vision.
2. **Air-gapped** — unplug the ethernet cable; Metis still works.
3. **Infinite memory** — ChromaDB + `.mts` identity matrix.
4. **Zero per-token cost** — flat $49/mo; user's GPU does the work.

## 8. Go-to-Market (Miami + Brickell)
- Launch in Miami's eMerge Americas 2026.
- Design-partner 5 Brickell hedge funds + 3 law firms for Enterprise tier.
- Developer funnel via Hacker News + GitHub open-source plugin ecosystem.

## 9. Financials
- Tier pricing — Free / Pro $20 / Enterprise $500.
- Marketplace 30% take.
- Target: 10k Free → 2k Pro → 50 Enterprise in year one = ~$600k ARR.

## 10. The Ask
**$2M seed to hit 10k installs and ship iOS + macOS companions.**
- 24-month runway, 6-person team.
- Board seat + observer for lead.
